# CycloFix 🌸

An adaptive menstrual pain prediction and management app for women aged 16–25.

CycloFix tracks your period pain, suggests personalised relief methods, and learns what actually works for you over time. The more you use it, the smarter it gets.

## Live App
🔗 [cyclofix-production.up.railway.app](https://cyclofix-production.up.railway.app)

## What it does
- **One-tap pain logging** — log intensity and pain type in under 10 seconds
- **Personalised relief suggestions** — heat, rest, movement, breathing tailored to your pain type
- **Adaptive feedback loop** — "did this help?" learns what works for you specifically
- **Cycle phase awareness** — know what phase you're in and what to expect each day
- **Period prediction warning** — 3-day heads up before your predicted period
- **Weekly personal insight** — one meaningful observation from your data every week
- **Pattern display** — pain timeline chart and what helped you most
- **PWA** — installable on Android and iPhone, works like a native app

## Tech stack
- **Backend:** Python + Flask
- **Database:** SQLite via Flask-SQLAlchemy
- **Auth:** Flask-Login
- **Frontend:** HTML, CSS, vanilla JavaScript
- **Deployment:** Railway
- **PWA:** Service worker + Web App Manifest

## Phase plan
- **Phase 1 (current):** Logging, suggestions, feedback loop, pattern display
- **Phase 2:** Pain severity prediction using physiological signals (HRV, temperature, sleep)
- **Phase 3:** Closed-loop smart relief device — adaptive heat/stimulation patch

## Vision
CycloFix is the first adaptive period pain management system that learns what works for each individual user. The long term vision is a complete ecosystem — software prediction layer feeding a smart wearable relief device — making menstrual pain manageable and predictable for every woman.
